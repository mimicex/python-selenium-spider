# chromedriver

請去 chromedriver 下載你版本的 chromedriver

```
https://chromedriver.chromium.org/downloads
```

放在預設的路徑或你要的路徑
```
sudo cp -R '檔案路徑' '/usr/bin/chromedriver'
```
更改權限
```
chmod +x '/usr/bin/chromedriver'
```

執行
```
python3 main.py
```
